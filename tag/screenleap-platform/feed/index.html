Screenleap Platform – Screenleap Blog https://blog.screenleap.com Live-Collaboration Platform for Your Website Tue, 11 Jul 2017 08:16:05 +0000 en-US hourly 1 182819694 Next Generation of Screenleap Platform Now Available for API https://blog.screenleap.com/2017/07/11/next-generation-screenleap-platform-now-available-api/ https://blog.screenleap.com/2017/07/11/next-generation-screenleap-platform-now-available-api/#respond Tue, 11 Jul 2017 16:00:24 +0000 https://blog.screenleap.com/?p=1889 We are excited to announce that the next generation of the Screenleap Platform is now available for the API. Over the last year, the Screenleap Platform has been completely rebuilt from the ground up to make it faster, more scalable, and more reliable. Please see our previous blog article for more information about the benefits of the new platform.

You can switch your screen shares to use the new platform by adding the “useV2=true” parameter to your request to create a screen share:

POST https://api.screenleap.com/v2/screen-shares?accountid=[account_id]&useV2=true

All API screen shares will default to the new platform starting on September 1st, but we recommend that you update your code to take advantage of the benefits now.

Please contact us at support@screenleap.com if you have any questions.
